# Circle Completation

## Props

### 1. value

- description: Completation value, can be Integer (0 ~ 100) or Float (0 &lt;= value &lt;= 1)
- type: `number`

### 2. size

- description: Sizes
- type: `'big' | 'medium' | 'small'`

### 3. image

- description: Adds an image to the center on the circle
- type: `string`

## Exemple

```jsx
<CircleCompletation value={10} />
```
